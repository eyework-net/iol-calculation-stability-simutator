# IOL Calculation Stability Simulator

眼内レンズ（IOL）度数計算における「予測の安定性」や「計算条件による変動しやすさ」を、視覚的に学べる教育用シミュレーターです。
https://eyework-net.github.io/iol-calculation-stability-simulator/
本ツールは、眼軸長（AL）、前房深度（ACD）、角膜屈折力（K値）、水晶体厚（LT）、角膜乱視量（CYL）、角膜横径（WTW）などの生体計測値を調整しながら、IOL度数計算ターゲットがどのような条件で安定しやすいか、あるいは注意が必要になるかを直感的に学習できるよう設計されています。

---

## 開発目的

白内障術前検査では、生体計測装置や計算式の進歩により、高精度なIOL度数計算が可能となりました。
一方で、数値をそのまま受け取るだけではなく、

* 測定値に整合性があるか
* 非典型的な眼球プロポーションではないか
* なぜ予測誤差が生じやすいのか
* 計算式ごとの差が出やすい症例ではないか

といった視点で結果を評価する力も重要です。

本ツールは、そのような**「数値を読み解く力」**を養うことを目的とした教育教材です。

---

## 主な機能

* 生体計測値のスライダー入力
* IOL計算安定性スコア表示
* 条件変化によるリスク要因の可視化
* 教育用コメント表示
* 各種ケーススタディ用途
* 学習者向けの直感的UI

## 使用方法

1. https://eyework-net.github.io/iol-calculation-stability-simutator/ をブラウザで開いてください。
2. 各パラメータを変更すると結果がリアルタイムで更新されます。
3. 条件による変化や各因子の影響を観察してください。

---

## 注意事項

本ツールは教育目的の概念モデルです。
実際のIOL度数決定や臨床判断を代替するものではありません。

実臨床では、各種IOL計算式、実測値、術者経験、患者背景などを総合的に考慮してください。
